# Mramba Edison | Personal Portfolio

A professional, dynamic, and visually appealing personal portfolio website. Built from scratch to showcase my skills as a Full Stack Developer, Wilcom embroidery digitizer, graphic designer, and creative technologist.

## 🚀 Features

- **Responsive Design**: Adapts beautifully to mobile, tablet, and desktop screens using modern CSS media queries.
- **Dynamic Animations**: Smooth CSS fade-in animations on scroll for an engaging user experience.
- **Modern UI Aesthetics**: Utilizes glassmorphism, gradient text highlights, background blur blobs, and premium dark mode styling.
- **Interactive Navigation**: Sticky, glass-effect navigation bar with a smooth scroll experience and a mobile-friendly hamburger menu.

## 🛠️ Built With

- **HTML5**: Semantic and structured markup.
- **CSS3**: Vanilla CSS, including Flexbox, CSS Grid, Custom Properties (variables), and Keyframe animations. No external frameworks used.
- **JavaScript**: Vanilla JS for DOM manipulation, interactive mobile menus, and smooth scrolling capabilities.
- **Icons & Typography**: FontAwesome for social/skill icons, and Google Fonts (`Inter` and `Outfit`) for modern, elegant typography.
